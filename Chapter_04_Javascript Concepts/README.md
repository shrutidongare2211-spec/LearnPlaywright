# Chapter 04: JavaScript Concepts

This folder covers important JavaScript concepts including variable declarations, functions, scope, and hoisting.

## Files

| File | Description |
|------|-------------|
| `09_var_let_const.js` | Introduction to `var`, `let`, and `const` |
| `10_functions.js` | Function declarations and usage |
| `11_var_explained.js` | Deep dive into `var` |
| `12_let_people_love.js` | Understanding `let` |
| `13_const_explained.js` | Understanding `const` |
| `14_var_functionscope.js` | Function scope with `var` |
| `15_let_scope.js` | Block scope with `let` |
| `16_Hoisting.js` | Concept of hoisting |
| `17_hoisting_fn.js` | Hoisting with functions |

## Key Topics

- `var`, `let`, and `const` differences
- Redeclaration and reassignment rules
- Function scope vs block scope
- Hoisting of variables and functions
- Best practices for declaring variables
