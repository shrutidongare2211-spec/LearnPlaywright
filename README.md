# Learn JavaScript

This repository contains my personal notes, examples, and exercises while learning JavaScript from scratch.

## Repository Structure

| Folder | Description |
|--------|-------------|
| `Chapter_01_Basics` | Introduction to JavaScript, setup verification, and basic syntax |
| `Chapter_02_Javascript_Commands` | Core JavaScript commands and fundamentals |
| `Chapter_03_Identifier_Literals` | Rules for identifiers, literals, comments, and VS Code shortcuts |
| `Chapter_04_Javascript Concepts` | Variables (`var`, `let`, `const`), functions, scope, and hoisting |
| `Chapter_05_Literal` | JavaScript literals: strings, template literals, numbers, `null`, `undefined` |
| `Chapter_06_Double_Triple_Equal` | JavaScript operators: arithmetic, comparison, logical, ternary, `typeof`, `??`, `++`/`--` |

## Topics Covered

- JavaScript basics and setup
- JavaScript commands
- Identifier naming rules
- Comments and literals
- `var`, `let`, and `const`
- Function declarations
- Scope and hoisting
- Strings, single/double quotes, and backticks (template literals)
- Number literals, BigInt, `Infinity`, `NaN`
- `null` and `undefined`
- Comparison operators: `==` vs `===`
- Confusing equality cases in JavaScript
- VS Code keyboard shortcuts for Windows

## How to Run

1. Clone the repository:
   ```bash
   git clone https://github.com/shrutidongare2211-spec/LearnPlaywright.git
   ```
2. Open any `.js` file in VS Code or your preferred editor.
3. Run the file using Node.js:
   ```bash
   node filename.js
   ```
