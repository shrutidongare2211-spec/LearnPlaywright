# Chapter 05: Literals in JavaScript

This folder covers different types of literals in JavaScript, including strings, numbers, template literals, `null`, and `undefined`.

## Files

| File | Description |
|------|-------------|
| `22_Literal.js` | Introduction to literals in JavaScript |
| `23_null_undefined.js` | Understanding `null` and `undefined` |
| `24_null.js` | Deep dive into `null` |
| `25_Literal_All.js` | Overview of all literals |
| `26_Literal_Number_all.js` | All number types: integers, floats, BigInt, `Infinity`, `NaN` |
| `27_String.js` | Strings with single and double quotes |
| `28_Template_Literal.js` | Template literals with backticks and variable interpolation |
| `29_Backtick_single_double.js` | Difference between single quotes, double quotes, and backticks |

## Key Topics

- String literals: `'Hello'` vs `"Hello"`
- Template literals: `` `Hello ${name}` ``
- Number literals: integers, floats, binary, octal, hexadecimal
- BigInt for large numbers
- Special values: `Infinity`, `-Infinity`, `NaN`
- `null` vs `undefined`
- Practical use cases in Playwright automation
