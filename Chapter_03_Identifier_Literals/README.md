# Chapter 03: Identifier Rules, Literals, and Comments

This folder explains the rules for naming identifiers in JavaScript, along with literals and comments.

## Files

| File | Description |
|------|-------------|
| `06_Identifier_Rules.js` | Valid and invalid identifier naming examples |
| `07_Identifier_Part2.js` | More identifier rules and examples |
| `08_Comments.js` | Single-line and multi-line comments in JavaScript |
| `js_Identifier_RulesByOpenCode.js` | Additional identifier rules reference |
| `VS_Code_keyboard_shortcut_windows.md` | Useful VS Code keyboard shortcuts for Windows |

## Key Topics

- Valid identifier names
- Use of `$` and `_` in identifiers
- Case sensitivity (`Name` vs `name`)
- Comments: `//` and `/* */`
- VS Code shortcuts for faster coding
