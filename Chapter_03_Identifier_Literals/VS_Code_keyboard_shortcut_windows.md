## Notes

- `Ctrl + K` means press and release `Ctrl + K`, then press the next key shown.
- Some shortcuts may conflict with extensions or custom keybindings.
- To view or customize every shortcut, open the Keyboard Shortcuts editor with `Ctrl + K Ctrl + S`.
